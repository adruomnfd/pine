## work-in-progress

### clone the repo:
```
git clone --recursive https://github.com/adruomnfd/pine
```

### build with CMake:
```
cmake -Bbuild
cmake --build build
```

### usage:
```
build/pine scenes/nspheres.txt result.bmp
```
then you can find the result `result.bmp` in the same folder
